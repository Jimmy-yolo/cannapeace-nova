## Troubleshooting

### Problem: "File not found" error

**Solution:**
- Check your 3 Excel files are in `C:\Reports\Input\`
- Check file names are exactly: `sales.xlsx`, `inventory.xlsx`, `expenses.xlsx` (case matters!)

### Problem: Script runs but report looks wrong

**Solution:**
- Make sure Excel files have the same column structure as before
- If you changed the format, contact Jimmy (this is included support)

### Problem: Script won't start

**Solution:**
- Restart your computer and try again
- If still doesn't work, take a photo of the error message and send to Jimmy

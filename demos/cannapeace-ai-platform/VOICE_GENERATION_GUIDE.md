## 📝 Step-by-Step: ElevenLabs (RECOMMENDED)

### Step 1: Sign Up
1. Go to https://elevenlabs.io
2. Click "Get Started Free"
3. Sign up with email/Google
4. Verify email

### Step 2: Generate Thai Greeting
1. Click "Speech Synthesis" (left menu)
2. Settings:
   - Model: Eleven Multilingual v2
   - Voice: Select "Rachel" or "Bella"
3. Language: Thai
4. Text box: Paste Thai script:
   ```
   สวัสดีค่ะ! ยินดีต้อนรับสู่แคนนาพีซ
   ร้านกัญชาคุณภาพพรีเมี่ยมที่ดีที่สุดในกรุงเทพ
   มีอะไรให้ช่วยคะวันนี้?
   ```
5. Voice Settings:
   - Stability: 50%
   - Clarity: 75%
   - Style Exaggeration: 0%
6. Click "Generate"
7. Play to preview
8. Download as MP3
9. Convert to M4A (online tool or rename)

### Step 3: Repeat for Other Languages
- English: Paste English script
- Chinese: Paste Chinese script
- Russian: Paste Russian script
- Japanese: Paste Japanese script
- Korean: Paste Korean script
- French: Paste French script

### Step 4: Download & Rename
```
greeting_thai.m4a
greeting_english.m4a
greeting_chinese.m4a
greeting_russian.m4a
greeting_japanese.m4a
greeting_korean.m4a
greeting_french.m4a
```

### Step 5: Upload to Railway
```bash
railway up
# Select greeting_thai.m4a
# Upload as: greeting_voice.m4a
```

**DONE!** Voice greeting will work instantly.

## 🎨 Voice Customization Tips

### For "Sweet, Young, Energetic" Voice:

**In ElevenLabs:**
- Use "Rachel" (most energetic)
- Or "Bella" (sweet, friendly)
- Stability: 40-60% (more variation = more natural)
- Clarity: 70-80% (clear but not robotic)

**In Google Cloud:**
- Choose "Neural2" voices (newest, most natural)
- Female voices ending in "A", "B", or "C"
- Thai: th-TH-Neural2-C
- Chinese: cmn-CN-Wavenet-A

**In Play.ht:**
- Filter: "Conversational" style
- Age: "Young adult"
- Energy: "Energetic"

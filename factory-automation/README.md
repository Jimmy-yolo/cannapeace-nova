## Selection Criteria

Pick the one that:
1. **Is genuinely painful** (people complain about it)
2. **Has measurable time cost** (X hours/week wasted)
3. **Is straightforward to automate** (can be done in 3-5 days)
4. **Has clear before/after** (easy to document the improvement)

**Do NOT pick:**
- Something that "might be nice" (pick real pain)
- Complex integrations with legacy systems (too slow)
- Anything requiring hardware (keep it software)
